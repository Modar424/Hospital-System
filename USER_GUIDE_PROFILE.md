# 📱 دليل استخدام ميزة الملف الشخصي

## 🎯 ماذا تم الإضافة؟

تم إضافة **تاب جديد في لوحة تحكم المريض** لعرض الملف الشخصي الكامل مع جميع البيانات والمعرّف الفريد.

---

## 🚀 كيفية الاستخدام

### 1️⃣ دخول لوحة التحكم
```
المريض يسجل الدخول
    ↓
ينتقل إلى /appointments
```

### 2️⃣ اختيار تاب الملف الشخصي
```
يظهر 3 تابات:
├── الحجوزات (Appointments)
├── الفواتير (Invoices)
└── ملفي (My Profile) ← اختر هنا
```

### 3️⃣ عرض البيانات
سيشاهد المريض:

#### 🔝 القسم الأول: معلومات أساسية
- اسم المريض
- البريد الإلكتروني
- **رقم الملف (Patient ID)** 📋
  - يمكن نسخه بزر واحد
  - يظهر تأكيد عند النسخ

#### 📊 الشبكة الثانية: البيانات الشخصية
| الحقل | المعلومة |
|------|---------|
| 📱 الهاتف | رقم الاتصال |
| 🎂 تاريخ الميلاد | YYYY-MM-DD |
| 👤 الجنس | ذكر/أنثى/آخر |
| 🩸 فصيلة الدم | O+, A-, وغيره |

#### 📍 معلومات الاتصال
- **العنوان**: كامل العنوان السكني
- **جهة اتصال الطوارئ**: الاسم والهاتف

#### 🏥 السجل الطبي
- **التاريخ الطبي**: قائمة بالأمراض السابقة
- **الحساسيات**: (باللون الأحمر للتنبيه)
- **ملاحظات**: أي معلومات إضافية

---

## ⚙️ الميزات المتاحة

### 💾 نسخ رقم الملف
```
الخطوة 1: انظر إلى رقم الملف (Patient ID)
الخطوة 2: اضغط على زر النسخ (📋)
الخطوة 3: سيظهر تأكيد "تم نسخ الرقم"
الخطوة 4: الرقم جاهز للصق في أي برنامج
```

### ✏️ تحديث البيانات
```
الخطوة 1: اضغط على زر "تحديث البيانات"
الخطوة 2: سيأخذك للصفحة /patient-profile
الخطوة 3: عدّل البيانات المطلوبة
الخطوة 4: احفظ التغييرات
```

## 🎨 واجهة المستخدم

### الألوان والرموز

```
🟦 الأزرق     → المعلومات الأساسية
🟨 الأصفر     → جهة الطوارئ (تحذير)
🟥 الأحمر     → الحساسيات (خطر)
🟦 البنفسجي   → الملاحظات
🟦 الأزرق الفاتح → التاريخ الطبي
```

### الحالات الخاصة

#### ✅ لديه ملف شخصي
```
سيشاهد جميع البيانات مع ملء كامل
```

#### ⚠️ بدون ملف شخصي
```
سيشاهد رسالة:
"لم تقم بإنشاء ملفك الشخصي"

مع زر:
"إنشاء ملفي الشخصي"
```

#### ⏳ جاري التحميل
```
سيشاهد تأثير تحميل (Skeleton Loader)
```

جميع البيانات تُحدّث تلقائياً عندما:

1. ✏️ يقوم المريض بتحديث ملفه
2. 🔄 ينتقل إلى تاب آخر ثم يعود
3. 🔃 يقوم بتحديث الصفحة

---

## 🐛 استكشاف الأخطاء

### المشكلة: "لا أرى ملفي الشخصي"

**الحل:**
1. تحقق من تسجيل دخولك
2. انتقل إلى `/patient-profile` لإنشاء الملف
3. احفظ البيانات
4. عد إلى `/appointments` وانقر على "ملفي"

### المشكلة: "البيانات تبدو قديمة"

**الحل:**
1. احفظ التحديثات في `/patient-profile`
2. انتظر ثانية أو اثنتين
3. قم بتحديث الصفحة (F5)

### المشكلة: "نسخ الرقم لا يعمل"

**الحل:**
1. تأكد من إتاحة الوصول إلى الحافظة
2. جرّب متصفح مختلف
3. استخدم نسخة محدثة من المتصفح

---

## 💡 نصائح مفيدة

1. **احفظ رقم الملف** - قد تحتاجه للاتصال بالمستشفى
2. **حدّث معلوماتك دورياً** - خاصة الحساسيات والأدوية
3. **تحقق من جهة الطوارئ** - تأكد من صحتها
4. **لاحظ التحذيرات** - الحساسيات تظهر بلون أحمر
